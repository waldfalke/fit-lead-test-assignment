# Fit&Lead Design System 🚀

Welcome to the **Fit&Lead** component library and design system documentation.

This Storybook showcases all reusable UI components used in the Fit&Lead test assignment project.

## 🎯 What's Inside

### Components
- **Button** - 7 variants, 5 sizes, multiple states
- **Card** - 3 variants for different use cases
- **Navigation** - Reusable navigation for Header, Footer, and Mobile
- **ThemeToggle** - Light/Dark mode switcher
- **Header** - Site header with logo and navigation
- **Footer** - Site footer with links and social media

### Design Tokens
- **137 design tokens** driving all styles
- **Colors**: Fit&Lead brand (#00484F primary, #FBEA1B accent)
- **Spacing**: 8px-based system
- **Typography**: 6-level responsive scale
- **Themes**: Light & Dark mode support

## 💡 Key Features

### Component Reusability
Every component shown here is used on **both pages** of the site (Landing + UI Kit), demonstrating real-world reusability.

### Contract-Driven Development
All components were created using detailed YAML contracts defining their API, variants, and requirements **before** writing code.

### Theme Support
All components are theme-aware and automatically adapt to light/dark themes using CSS variables.
